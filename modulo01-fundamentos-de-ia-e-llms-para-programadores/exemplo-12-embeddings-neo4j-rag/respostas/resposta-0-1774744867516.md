Tensors são a **base fundamental** da TensorFlow.js e representam os dados que os algoritmos de machine learning processam. Think of them como **matrizes multidimensionales de números**, que podem ser simples (como um único número) ou complexas (como imagens ou séries temporais).

### Como são representados em JavaScript:
1.  **Estrutura de Dados:** Em JavaScript, tensors são criados usando a biblioteca `tfjs-core`. Eles são **objetos imutáveis** (não podem ser alterados após criados) e contêm:
    *   **Valores numéricos:** Todos os elementos são números (float ou int).
    *   **Forma (Shape):** Uma tupla indicando as dimensões. Por exemplo:
        *   `(3,)` = um vetor de 3 elementos (1D).
        *   `(2, 3)` = uma matriz 2x3 (2 linhas, 3 colunas - 2D).
        *   `(1, 2, 3)` = um tensor 3D (1 plano, 2 linhas, 3 colunas).
    *   **Tipo (DType):** O tipo de número (ex: `tf.float32`, `tf.int32`, `tf.bool`).

2.  **Criação em JavaScript (Exemplo do Contexto):**
    O exemplo mostrado converte dados de pessoas (que são objetos) em um tensor numérico. Isso é crucial porque TensorFlow.js **não entende objetos ou strings diretamente**.

    ```javascript
    // Dados originais (objetos)
    const pessoas = [
      { nome: "Erick", idade: 30, cor: "azul", localizacao: "São Paulo" },
      { nome: "Ana", idade: 25, cor: "vermelho", localizacao: "Rio" },
      { nome: "Carlos", idade: 40, cor: "verde", localizacao: "Curitiba" }
    ];

    // Convertendo para um tensor numérico
    const tensorPessoas = tf.tensor([
      [30, 1, 0, 0],  // Erick: idade=30, cor=azul (1), localizacao=SP (0)
      [25, 0, 1, 0],  // Ana: idade=25, cor=vermelho (1), localizacao=Rio (0)
      [40, 0, 0, 1]   // Carlos: idade=40, cor=verde (1), localizacao=Curitiba (1)
    ], [3, 4]);       // Formato: 3 pessoas (linhas) x 4 features (colunas: idade, cor1, cor2, cor3)
    ```

    *   **Forma:** `[3, 4]` indica 3 linhas (pessoas) e 4 colunas (features: idade + 3 colunas para as 3 cores).
    *   **Valores:** Todas as entradas são números inteiros.
    *   **DType:** `tf.int32` (ou `tf.float32` dependendo do uso).

### Analogia Prática:
Imagine você tem um **excel com dados de pessoas**. Cada linha é uma pessoa, e cada coluna é uma característica (idade, cor preferida, localização). Um tensor é como **copiar esse excel para um arquivo de texto plano** (CSV), onde todos os dados são representados apenas por números e linhas separadas por vírgulas. TensorFlow.js precisa desse "arquivo texto plano" numérico para entender os dados.

### Por que isso é importante?
TensorFlow.js processa **apenas dados numéricos**. Se você tem dados não numéricos (como nomes de cores ou strings), você **DEVE converter** esses dados em números (como feito com as cores azul/vermelho/verde) antes de criar o tensor. Isso é chamado de **preprocessamento de dados**.

Em resumo: Tensores são **matrizes de números** que representam dados em TensorFlow.js. Em JavaScript, eles são criados usando a biblioteca `tfjs-core` e são essenciais para que o framework entenda e processe dados para aprendizado de máquina.
